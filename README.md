# Firebase Studio

This is a NextJS starter in Firebase Studio.

## Entry Point

The main entry point of the application is `src/app/(app)/dashboard/page.tsx`.

## Running Locally

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

## Building

Create an optimized production build with:

```bash
npm run build
npm start
```

## Deployment

Deploy the production build to Firebase App Hosting using the Firebase CLI:

```bash
firebase deploy
```

Ensure you have the Firebase CLI installed and are logged in before deploying.
